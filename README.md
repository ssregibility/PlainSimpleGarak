# Plain Simple Garak - A Quote Generator

Plain Simple Garak is a GRU quote generation model - 

## Requirements

- Python 3.6.9+
- Pytorch 1.7.0+
- CUDA 10.1+
- nltk 3.0+
- Pandas 1.0+

To install requirements:

```setup
pip3 install -r requirements.txt
```
